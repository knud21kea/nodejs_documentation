Node.jsÂ® is an open-source, cross-platform JavaScript runtime environment.

 Node is javascript taken out of the browser so it becomes backend, so we can make server software, scripts, gui etc (independent of a browser) and without having to use another language for the backend development.


To navigate the site:

 Each icon in the top menu points at an endpoint in the backend which serves one page. Hovering over an icon will give a pop up showing the main theme of that page. Click to load that page. The light bulb icon requests that display mode is toggled between dark (default) and light. This will however return to this page when used on others.
