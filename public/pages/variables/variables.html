Variables and data types

To be able to operate on variables, it is important to know something about the type.

 We will look at data types and type coercion in JavaScript. 'var' can be used to specify the data type, but we try to minimise its use as it has global scope which can be difficult to debug.
 We use 'const' as much as possible, which does need to be assigned a value when declared and otherwise 'let' for local variables with changing values.


Data types:

There are 8 data types in javascript

  • String - represents sequence of characters
  • Number - represents numeric values
  • Bigint - represents integers with arbitrary magnitude
  • Boolean - represents boolean value
  • Undefined - represents undefined value
  • Null - represents no value
  • Symbol - is a unique and immutable primitive value and may be used as the key of an Object property
  • Object (object, array, date) - the only mutable values. Functions are also objects

Dynamic and weak typing:

JavaScript which is a dynamic language with dynamic types. Variables in JavaScript are not directly associated with any particular value type and any variable can be assigned (and re-assigned) values of all types. 

        
		let myVariable = 7; // myVariable is now a number
		myVariable = "a prime number"; // myVariable is now a string
		myVariable = true; // myVariable is now a boolean

JavaScript is also a weakly typed language, which means it allows implicit type conversion when an operation involves mismatched types, instead of throwing type errors. 

        
		const myVariable = 7; // a number
		const result = myVariable + "1"; // JavaScript coerces myVariable to a string
		console.log(result); // "71"

Type coercion:

JavaScript is a weakly typed language. This means that you can often use a value of one type where another type is expected, and the language will convert it to the right type for you. To do so, JavaScript defines coercion rules.

  • Primitive coercion - where a primitive value is expected, but no strong preference for the actual type
  • Numeric coercion - bigint, number
  • Other coercions - string, boolean, object

The details of coercion rules are beyond the scope of this document.
